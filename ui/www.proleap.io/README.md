proleap-sales
=============

Code base for ProLeap website. This is a single page application (SPA).


Init
----

``` bash
$ brew install node
$ npm install -g @vue/cli
$ npm install -g @vue/cli-service
```

``` bash
$ npm install
$ npm run serve
$ npm run build
```


Serve
-----

``` bash
$ npm install
$ npm run serve
```


Build
-----

``` bash
$ npm run build
```


Vue Upgrade
-----------

``` bash
$ vue upgrade
```


Firebase
--------

``` bash
$ npm install -g firebase-tools
$ firebase login
$ firebase init
```


Deploy
------

``` bash
$ firebase deploy
```
